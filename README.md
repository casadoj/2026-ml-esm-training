# Machine Learning for Earth Systems Modelling

Code examples from ECMWF's Machine Learning for Earth Systems Modelling online courses, run over 2026. Their primary intention is to support the learning in the online course [Machine Learning for Earth Systems Modelling: Architectures, Data and Prediction](https://learning.ecmwf.int/course/view.php?id=99). Enrol in this (free) course for context on the Jupyter notebooks. Also note that the notebooks are not officially supported by ECMWF, and that this repository will be in development while the course is under construction.
